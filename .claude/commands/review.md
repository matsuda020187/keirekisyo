description: code-reviewer subagent で変更をレビューし must 指摘を解消する
---

<!--

役割: code-reviewer subagent の呼び出しを定型化。
「レビューして」の言い方の揺れに依存せず、確実に独立コンテキストの
レビュアーに委譲させる。指摘への対応もセットで指示しておくことで、
「レビューしっぱなし」を防ぐ。
-->

code-reviewer subagent を使って、現在の変更内容をレビューしてください。

レビュー結果を受け取ったら:

1. must の指摘はすべて修正する
2. should の指摘は、修正するか見送るかを理由付きで判断して報告する
3. nit は対応不要(一覧だけ残す)
4. 修正を行った場合は npm run verify を再実行し、
   必要なら再度レビューに出す(must がゼロになるまで)
